Gridea同步失败问题

大家在使用Gridea的时候会不会也出现同步失败的情况，有的话看这里！！！


这种情况多半是系统代理导致的，可以试试以下的方法

方法一：
梯子（魔法就不教了）

方法二：

  1. 关闭加速器，steam++，clash之类的网络代理软件（包括翻墙软件和VPN）

  2. win+R输入cmd进去命令指令行

  3. 分别复制粘贴运行以下命令

npm config delete proxy

npm config delete https-proxy

netsh winhttp reset proxy

方法三：
用steam++，火绒内置的火绒断网修复或其它工具重置host文件（或手动删除host文件）

重启电脑后99%的同步失败问题就都能解决了

当然也可以试试这位大佬的方法（只不过可能会很麻烦）
